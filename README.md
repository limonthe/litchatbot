### 上手指南
1. 输入问题
2. 等待机器人响应回答



###### 配置要求
1. Windows/Linux/Mac
2. Chorme内核浏览器
3. 稳定网络环境

###### API_KEY获取
1. 注册智谱开放平台 [https://www.zhipuai.cn/](https://www.zhipuai.cn/)
2. 在“个人面板”处查询api_key
3. 注：demo已内置api_key以供体验
